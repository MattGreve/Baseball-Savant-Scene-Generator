# Baseball Savant Scene Generator

## What It Does

Sorts user input data from baseball savant and turns it into a 3D animated bar graph. It will support data input for hitting or pitching. It will sort data input from a copy paste automatically and then build the scene automatically.

## Project Structure

Savant_Graph_Scene_Generator
	Geometry_utils.py	# Geometry creation functions
	Material_utils.py	# Material creation functions based on stats
	main.py	# Main execution file

## Features

- [ ] Automatic Baseball Savant data parsing
- [ ] Procedural bar graph generation
- [ ] Support for hitting statistics
- [ ] Support for pitching statistics
- [ ] Custom geometry builder functions
- [ ] Material assignment based on stat values
- [ ] Automatic object positioning
